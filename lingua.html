Sobre as linguagens que eu sei usar . . .

JAVA

A linguagem java foi a primeira que eu aprendi a entrar no curso tecnico, sendo uma que eu fui indo para grandes desafios, alem dos que o professor ensinava, como quando os outros estavam aprendendo "OptioPane", eu estava vendo "JFrame".
Minha maior dificuldade era que eu fazia coisas avançadas, porem, eu muita das vezes, não conseguia fazer coisas que estavam debaixo do meu nariz.

HTML

A linguagem HTML foi a segunda linguagem que tive contato para programar, fazendo coisas simples como por texto e imagem, porem agora no curso, sei fazer coisas mais coisas alem de por um simples texto e imagem.
Minha maior dificuldade é que acabo fazendo o codigo de forma desorganizada, fazendo com que fique dificil de achar algo.

CSS

A linguagem CSS foi a linguagem que eu aprendi junto com o HTML, porem nela eu não sabia fazer nada, então eu não conseguia fazer coisas extravagantes, porem agora no curso eu sei fazer multiplas coisas, até mesmo coisas que não foi ensinada e sim buscadas por conta própria.
Minha maior dificuldade é acabar criando classes que são desnecessarias, fazendo meu codigo ficar extremamente grande.

Scratch

A linguagem Scratch foi a primeira na qual eu tive contato, antes mesmo do curso, nesta linguagem era algo bem simples, ate mesmo em um ano do fundamental do meu colegio o sistema EV3 teve uma atualização extremamente parecida com essa linguagem, e sendo uma linguagem muito simples, eu não possuia dificuldades de fazer algo.
